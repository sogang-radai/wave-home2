- WaveAI버튼을 topbar에 검색창처럼 좀 더 넓게 배치해서 누르기 편하게 해보면 어떨까? -> 쉽지는 않다

## 주간 계획탭 관련
- 'AI 맞춤 추천 계획'이 '오늘의 할일'에 비해서 너무 열이 많음 -> UI 재배치
- 'AI 맞춤 추천 계획' 관련
    - UI가 좀 높아보임
    - '오늘의 권장 액션 • 에어컨 예약을 새벽 4시까지 1시간 연장' 처럼 한 열에 몰아쓰기
    - 승인 버튼을 오른편에 정렬
    - '승인됨'목록에서 '적용됨'버튼을 호버링하면 빨간색'제거'버튼으로 변경
    - 승인을 사용자가 하기 싫으면 지울수도 있는 UI추가
- '계획 추가'버튼 관련
    - 상단에 버튼형식으로 배치x
    - 계획을 주간 중 어디에 배치할지 모름 -> 날짜 선택 필요
    - 자세 수면 식습관 멘탈 중 카테고리를 선택하는건 에이전트가 알아서 한다는 느낌 -> 콤보박스 제거
    - '계획 추가' 버튼을 넣지 말고, PC기준으로 각 요일 별 빈 공간에 호버링 하면 클릭할 수 있다는 +모양 UI 제공
    - 팝업으로 1열짜리 텍스트 인풋 말고 좀 더 넓은 인풋을 제공
    - 추가 버튼을 인풋 내부에 둥근 엔터모양 버튼으로 -> 이건 알아서
    - 엔터 누르면 버튼 안눌러도 추가가 되도록
