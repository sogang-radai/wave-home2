# WaveHome
wave home 프로젝트는 SW/AI 중심대학 디지털 경진대회 출품을 목적으로 제작되는 "에이전트 연동 종합 라이프스타일 혁신" 애플리케이션입니다.

### 자세 교정
### 가전 제어

## 기기
애플리케이션에 연동되는 디바이스는 입력 장치와 출력 장치로 구분됩니다.

구동 장치
- raspberry pi5
- x64 ubuntu PC(개발용)

입력 장치
- 레이더 센서(srs r4sn 모델)
- 카메라
- esp32 + i2s 마이크

출력 장치
- TV
- IoT Plug
- Hue
- 에어컨(IR 방식)
- IR 리모트
